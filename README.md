# LitJson Unity

LitJson v0.19.0，打包为 UPM 格式。

## 安装

在 Unity 项目的 `Packages/manifest.json` 中添加：

```json
{
  "dependencies": {
    "com.veryfreestyle.unity.litjson": "https://github.com/veryfreestyle/litjson-upm.git"
  }
}
```

指定 commit 或 tag：

```json
"com.veryfreestyle.unity.litjson": "https://github.com/veryfreestyle/litjson-upm.git#v0.19.0"
```

## 要求

- Unity 2021.3+

## 原始来源

[LitJson](https://github.com/LitJSON/litjson) v0.19.0
